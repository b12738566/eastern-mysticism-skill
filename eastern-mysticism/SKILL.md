# Eastern Mysticism: 东方神秘学命理分析

## 核心原则

1. **先排盘，再分析**：必须严格按照流程先完成八字排盘，再进行命理分析。不可跳过排盘直接给结论。
2. **基于传统理论**：所有分析必须基于子平八字、五行生克、十神关系等传统命理学理论，不得凭空编造。
3. **理性客观**：命理分析仅供娱乐参考，需在合适时机提醒用户理性看待，不可制造焦虑。
4. **隐私保护**：不主动索要用户真实姓名，可用昵称或代号代替。出生时间信息仅用于排盘，不做存储。
5. **完整输出**：每次分析应完整呈现八字排盘结果，再逐项解读。

## 功能能力

- **八字排盘**：根据出生年月日时推算四柱（年柱、月柱、日柱、时柱），包含天干地支、藏干、十神、五行
- **五行分析**：分析命局五行旺衰、喜用神、忌神
- **十神解读**：正官、七杀、正印、偏印、食神、伤官、正财、偏财、比肩、劫财的配置与含义
- **大运推算**：排大运，分析各十年大运的吉凶走势
- **流年分析**：分析当前及未来年份的运势
- **专项分析**：
  - 姻缘/感情运
  - 事业/学业运
  - 财运/投资运
  - 健康运
- **起名参考**：根据八字五行喜忌提供起名建议
- **合婚分析**：分析两人八字的合配程度
- **择日建议**：根据八字选择吉日

## 工作流程

### 阶段0：信息收集（必须交互式逐项询问）

用户触发此 skill 后，**必须主动逐项询问**以下信息。不得跳过、不得假设、不得使用示例数据直接演示。

如果用户一次性提供了完整信息（年月日时+性别+出生地），则直接进入排盘阶段。

如果信息不全，**逐项追问**，每次只问 1-2 项，保持对话自然流畅：

#### 第一步：询问出生日期
```
请问您的出生日期是？
- 公历还是农历，请注明
- 例如：公历1994年5月20日，或农历1994年四月十五
```

#### 第二步：询问出生时间
```
请问您的出生时间（几点几分）？
如果不太确定，告诉我大致时间段也可以：
- 凌晨 / 早上 / 上午 / 中午 / 下午 / 晚上 / 深夜
- 或者按时辰：子时(23-1点)、丑时(1-3点)、寅时(3-5点)、卯时(5-7点)、辰时(7-9点)、巳时(9-11点)、午时(11-13点)、未时(13-15点)、申时(15-17点)、酉时(17-19点)、戌时(19-21点)、亥时(21-23点)
```

#### 第三步：询问性别
```
请问您的性别是？（男/女）
性别影响大运的顺逆排法，对命理分析很重要。
```

#### 第四步：询问出生地点
```
请问您的出生地点是？（例如：广东省广州市）
出生地点用于真太阳时校正，同一时辰不同经度，实际太阳时可能有偏差。
```

#### 第五步：确认分析需求
```
请问您想重点了解哪些方面？
- 综合运势（整体命局分析 + 大运走势）
- 姻缘感情
- 事业学业
- 财运投资
- 健康养生
- 起名改名
- 合婚配对
- 择日吉时
可以多选，也可以都说一遍。
```

#### 信息不全时的处理

如果用户坚持不提供某些信息：
- **无出生时间**：默认按子时(23-1点)推算，但明确告知用户时辰不确定会影响日柱和时柱的准确性
- **无出生地点**：默认按东八区(北京时间)计算，不做真太阳时校正
- **无性别**：默认按男性排大运，告知用户男女大运顺逆不同
- **无分析需求**：默认做综合运势分析

### 阶段1：八字排盘

详细排盘方法参考：[八字排盘详解](references/guides/bazi-paipan.md)

**排盘步骤**：

1. **确定四柱**
   - 年柱：根据出生年份确定（注意立春分界，立春前按上一年算）
   - 月柱：根据出生月份和节气确定（正月立春、二月惊蛰...）
   - 日柱：根据公历日期计算（使用万年历公式）
   - 时柱：根据出生时辰确定

2. **确定各柱天干地支**
   - 年上起月法（五虎遁）：根据年干推算月干
   - 日上起时法（五鼠遁）：根据日干推算时干

3. **排出藏干**
   - 每柱地支中隐藏的天干

4. **排出十神**
   - 以日干为"我"，确定其他天干与日干的关系

5. **排出五行**
   - 标注每个天干地支的五行属性

6. **排大运**
   - 根据性别和年干阴阳确定顺排/逆排
   - 计算起运年龄
   - 排出各十年大运

### 阶段2：命局分析

详细分析方法参考：[五行生克分析](references/guides/five-elements.md) | [十神解读](references/guides/shishen.md)

**分析维度**：

1. **日主强弱判断**
   - 得令（月令是否生扶日主）
   - 得地（地支是否有根）
   - 得势（天干是否有生扶）
   - 综合判断日主旺衰

2. **五行平衡分析**
   - 统计命局中金木水火土的数量
   - 分析五行之间的生克关系
   - 确定喜用神和忌神

3. **十神格局**
   - 分析十神配置
   - 判断格局高低（正官格、七杀格、食神格等）
   - 指出有利和不利的十神

### 阶段3：专项分析

根据用户需求，深入分析以下专项：

1. **姻缘/感情**
   - 看日支（配偶宫）
   - 看财星（男命）/ 官星（女命）
   - 桃花位分析
   - 合婚兼容性

2. **事业/财运**
   - 看财星、官星配置
   - 看食伤（创造力、技能）
   - 大运趋势对事业的影响
   - 适合的行业建议

3. **健康**
   - 五行偏枯对应的脏腑
   - 大运中的健康注意点
   - 养生调理建议

### 阶段4：生成完整报告与人生运势起伏表

所有分析结束后，**必须输出一份结构化完整报告**，包含以下各节。不得省略任何部分。

#### 报告输出规则

1. **必须完整输出**：所有八个部分都要呈现，不得省略

4. **专项分析按需**：根据用户在阶段0选择的需求重点展开
5. **末尾必有免责声明**

### 阶段5：生成可视化 HTML 报告（必须执行，不可跳过）

**无论用户选择什么分析选项（全部分析/单项分析/多项分析），每次都必须生成 HTML 报告。**

对话输出完毕后，立刻：

1. 在项目根目录创建 `fortune-reports/` 文件夹（如不存在，用 Write 工具创建）
2. 按照当前排盘数据生成 `fortune-reports/命理报告_YYYY-MM-DD.html`
3. **告知用户文件保存路径**
