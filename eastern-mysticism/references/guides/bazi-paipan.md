# 八字排盘详解

## 1. 年柱推算

年柱以立春为分界点，立春前按上一年干支计算。

### 公历年份 → 年干年支

使用公式计算：
- 年份减去3，除以10取余数（0为癸，1为甲，2为乙...9为壬）
- 年份减去3，除以12取余数（0为亥，1为子，2为丑...11为戌）

**示例**：1984年
- (1984-3)÷10 = 198余1 → 甲
- (1984-3)÷12 = 165余1 → 子
- 1984年为甲子年

**注意**：立春通常在2月4日前后。1月出生通常算前一年。

### 年柱速查（1900-2030年常用）

| 年份 | 干支 | 年份 | 干支 | 年份 | 干支 |
|------|------|------|------|------|------|
| 1980 | 庚申 | 1990 | 庚午 | 2000 | 庚辰 |
| 1981 | 辛酉 | 1991 | 辛未 | 2001 | 辛巳 |

| 1989 | 己巳 | 1999 | 己卯 | 2009 | 己丑 |

规律：每年干支顺推一位。

## 2. 月柱推算

月柱以节气为分界，对应十二地支固定不变：

| 月份 | 地支 | 节气分界 |
|------|------|----------|
| 正月 | 寅 | 立春 |
| 二月 | 卯 | 惊蛰 |
| 三月 | 辰 | 清明 |
| 四月 | 巳 | 立夏 |
| 五月 | 午 | 芒种 |
| 六月 | 未 | 小暑 |
| 七月 | 申 | 立秋 |
| 八月 | 酉 | 白露 |
| 九月 | 戌 | 寒露 |
| 十月 | 亥 | 立冬 |
| 冬月 | 子 | 大雪 |
| 腊月 | 丑 | 小寒 |

### 年上起月法（五虎遁）

根据年干确定正月（寅月）的天干：

| 年干 | 正月天干 | 口诀 |
|------|----------|------|
| 甲或己 | 丙 | 甲己之年丙作首 |
| 乙或庚 | 戊 | 乙庚之岁戊为头 |
| 丙或辛 | 庚 | 丙辛必定寻庚起 |
| 丁或壬 | 壬 | 丁壬壬位顺行流 |
| 戊或癸 | 甲 | 若问戊癸何方发，甲寅之上好追求 |

确定正月天干后，按天干顺序顺推其余月份。

**示例**：甲子年，月干从"丙"开始
- 正月寅月 = 丙寅
- 二月卯月 = 丁卯
- 三月辰月 = 戊辰
- ...类推

## 3. 日柱推算

日柱是最复杂的部分。使用基数和日期计算。

### 日干支计算公式

**方法一：通用公式**
```
1. 计算年份基数：
   对于1900-1999年：(年尾两位数+3)×5 + 55 + (年尾两位数-1)÷4
   对于2000-2099年：(年尾两位数+7)×5 + 15 + (年尾两位数+19)÷4
   
2. 计算该年该日的天数（从1月1日开始算）

3. (基数 + 天数) ÷ 60 取余数 → 日干支序号
   序号1=甲子，2=乙丑...60=癸亥
```

**方法二：使用已知日柱推算**
- 已知1900年1月1日为甲戌日（序号11）
- 用此基数加总天数取余即可

### 日柱速查思路

由于日柱精确计算需要万年历，实际使用时：
1. 用户给出公历日期 → 用日历工具或万年历直接查
2. 用户给出农历日期 → 需先转换为公历
3. 可使用编程方式计算（但skill中尽量手动推算或告知使用万年历）

### 日上起时法（五鼠遁）

根据日干确定子时的天干：

| 日干 | 子时天干 | 口诀 |
|------|----------|------|
| 甲或己 | 甲 | 甲己还加甲 |
| 乙或庚 | 丙 | 乙庚丙作初 |
| 丙或辛 | 戊 | 丙辛从戊起 |

| 戊或癸 | 壬 | 戊癸何方发，壬子是真途 |

确定子时天干后，按天干顺序顺推其余时辰。

**示例**：日干为甲，子时 = 甲子，丑时 = 乙丑，寅时 = 丙寅...

## 6. 大运排法

### 顺排/逆排判断

| 年干阴阳 | 性别 | 排法 |
|----------|------|------|
| 阳年（甲丙戊庚壬） | 男 | 顺排 |
| 阴年（乙丁己辛癸） | 男 | 逆排 |
| 阳年（甲丙戊庚壬） | 女 | 逆排 |
| 阴年（乙丁己辛癸） | 女 | 顺排 |

### 起运年龄计算

- **顺排**：从出生日顺数到下一个节气的天数，3天为1岁（精确到月）
- **逆排**：从出生日逆数到上一个节气的天数，3天为1岁

### 排大运

从月柱开始，根据顺逆方向，每柱管十年：
- **顺排**：月柱→下一柱→再下一柱...（按六十甲子顺序）
- **逆排**：月柱→上一柱→再上一柱...（按六十甲子逆序）

**示例**：
- 月柱为甲子，顺排大运：乙丑(10年)→丙寅(20年)→丁卯(30年)→戊辰(40年)→...
- 月柱为甲子，逆排大运：癸亥(10年)→壬戌(20年)→辛酉(30年)→庚申(40年)→...
