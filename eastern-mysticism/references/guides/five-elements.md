# 五行生克与命局分析

## 五行基本属性

### 木
- **方位**：东方
- **季节**：春
- **颜色**：青/绿
- **脏腑**：肝、胆
- **五官**：目
- **情志**：怒
- **味道**：酸
- **特性**：生长、条达、舒展
- **代表**：仁慈、生机、向上

### 火
- **方位**：南方
- **季节**：夏
- **颜色**：赤/红
- **脏腑**：心、小肠
- **五官**：舌
- **情志**：喜
- **味道**：苦
- **特性**：炎热、向上、明亮
- **代表**：热情、礼仪、文明

### 土
- **方位**：中央
- **季节**：长夏（季末18天）
- **颜色**：黄
- **脏腑**：脾、胃
- **五官**：口
- **情志**：思
- **味道**：甘
- **特性**：承载、化生、养育
- **代表**：诚信、稳重、包容

### 金
- **方位**：西方
- **季节**：秋
- **颜色**：白
- **脏腑**：肺、大肠
- **五官**：鼻
- **情志**：悲
- **味道**：辛
- **特性**：沉降、收敛、肃杀
- **代表**：义气、刚毅、果断

### 水
- **方位**：北方
- **季节**：冬
- **颜色**：黑/蓝
- **脏腑**：肾、膀胱
- **五官**：耳
- **情志**：恐
- **味道**：咸
- **特性**：润下、流动、寒凉
- **代表**：智慧、灵活、深沉

## 五行生克关系

### 相生（正向循环）
```
    木
   ↗  ↘
  水    火
  ↑      ↓
  金  ←  土
```

- **木生火**：木燃烧产生火（木是火之母，火是木之子）
- **火生土**：火燃烧后化为灰土
- **土生金**：土中蕴藏金属矿物
- **金生水**：金属遇冷凝结成水珠
- **水生木**：水滋养树木生长

### 相克（制约平衡）
```
    木
  ⤩    ⤡
 水      火
  ↑      ↓
  金  ←  土
```

- **木克土**：树木根系固定土壤
- **土克水**：土堤挡水
- **水克火**：水能灭火
- **火克金**：火能熔化金属
- **金克木**：金属能砍伐树木

### 反克（反侮）

当被克一方过强时，可反过来克制克方：
- 木本克土，土过重则木折 → **土反侮木**
- 土本克水，水过多则土崩 → **水反侮土**
- 水本克火，火过旺则水干 → **火反侮水**
- 火本克金，金过硬则火灭 → **金反侮火**
- 金本克木，木过强则金缺 → **木反侮金**

## 命局五行分析

### 1. 统计五行数量

将八字中所有天干、地支（含藏干）的五行属性统计出来。

每个地支的主气权重更高，藏干权重递减：
- 地支主气：权重 1
- 地支中气：权重 0.5
- 地支余气：权重 0.3

天干权重均为 1。

### 2. 判断日主旺衰

**得令**（月令支持）：
- 月支为日主同五行或生扶日主的五行 → 得令
- 例如：日主为甲木，生于寅卯月 → 得令

**得地**（地支根基）：
- 日支或其他地支中有日主同五行 → 得地
- 地支中有生扶日主的五行 → 半得地

**得势**（天干帮身）：
- 其他天干有同五行或生扶日主的 → 得势

综合判断：
- 得令+得地+得势 → 身旺（强）
- 只得其二 → 中等偏旺
- 只得其一 → 中等偏弱
- 全不得 → 身弱

### 3. 确定喜用神与忌神

| 日主状态 | 喜用神 | 忌神 |
|----------|--------|------|
| 身旺 | 克我、泄我（官杀、食伤、财星） | 生我、扶我（印星、比劫） |
| 身弱 | 生我、扶我（印星、比劫） | 克我、泄我（官杀、食伤、财星） |
| 专旺格（极旺） | 生扶为主，可顺不可逆 | 克制专旺五行的 |
| 从格（极弱） | 顺从旺势的五行 | 生扶日主的五行 |

### 4. 五行缺失与补救

**五行缺失的影响**：
- 缺木：缺乏仁慈心、决断力弱
- 缺火：缺乏热情、行动力不足  
- 缺土：缺乏诚信、稳定性差
- 缺金：缺乏果敢、义气不足
- 缺水：缺乏智慧、灵活性差

**补救方法**：
- 名字中补对应五行偏旁的字
- 穿戴对应颜色的衣物
- 从事对应五行的行业
- 居住/办公选对应方位
